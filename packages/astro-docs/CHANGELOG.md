# astro-docs

## 0.1.0

### Minor Changes

- aa72075: Initial public release. A customizable documentation **and book** framework for Astro: config-driven sidebar, Pagefind search, table of contents with scrollspy, callouts, prev/next, component overrides, a full design-token system with built-in themes, and book mode (numbered chapters, `@fig`/`@sec` cross-references, KaTeX). Bring your own CSS — framework styles live in cascade layers so yours always win.
