## Welcome

**astro-docs** is a documentation and book framework for Astro. It gives you a
sidebar, search, table of contents, callouts, and more — while letting you keep
your own look.

## Why another docs framework

Most documentation themes make your docs look like the framework. astro-docs
inverts that: framework styles live in CSS cascade layers, so your own
stylesheet always wins. The docs look like *your* site.

> [!NOTE]
> This page is itself rendered by astro-docs.

## Features

- Config-driven sidebar with autogenerate
- Build-time search via Pagefind
- Callouts, table of contents, breadcrumbs, prev/next
- Book mode with chapters, numbered figures, and cross-references
- Zero client JavaScript by default
